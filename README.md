# pupoo Platform Web Template (Vite + React + Tailwind v4)

## Run
```bash
npm install
npm run dev
```

## Public routes
- /
- /project
- /features
- /use-cases
- /tech
- /news

## Admin routes
- /admin
- /admin/events
- /admin/participants
- /admin/checkin
- /admin/payments
- /admin/notices
- /admin/community
- /admin/settings

## Tailwind v4
- Uses official @tailwindcss/vite plugin.
- Do NOT add tailwindcss into postcss.config.js.
