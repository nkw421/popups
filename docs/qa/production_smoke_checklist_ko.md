# 운영 전 최종 점검 체크리스트

1. 인증
- [ ] 새로고침 후 보호 페이지 접근 시 로그인 루프가 없는지
- [ ] 401 발생 시 refresh 1회만 수행되고 실패 시 정상 로그아웃되는지

2. 라우팅
- [ ] `/auth/login`은 비로그인 사용자만 접근 가능한지
- [ ] `/mypage`, `/mypage/*`는 인증 없이 접근 시 로그인으로 이동하는지

3. 결제
- [ ] `/registration/paymenthistory`가 세션 상태에 맞게 표시되는지
- [ ] `/payment/approve` 진입 시 부트스트랩 후 승인 로직이 실행되는지

4. 스토리지
- [ ] `/mypage/storage`에서 샘플 이미지 생성이 되는지
- [ ] 생성 파일이 실제 `/api/files` 업로드로 연결되는지
- [ ] 업로드 결과(`fileId`, `publicPath`)가 렌더링되는지

5. 빌드
- [ ] `npm run build` 성공
- [ ] 치명적 콘솔 에러(런타임) 없는지
