# Backend Compose Environment Example

이 문서는 `pupoo_backend`를 `docker-compose`에서 실행할 때 필요한 환경변수 예시를 정리한다.

## 필수

```env
SPRING_PROFILES_ACTIVE=prod
SERVER_PORT=8080

SPRING_DATASOURCE_URL=jdbc:mysql://host.docker.internal:3306/pupoodb?serverTimezone=Asia/Seoul&characterEncoding=UTF-8&useUnicode=true&useSSL=false&allowPublicKeyRetrieval=true
SPRING_DATASOURCE_USERNAME=pupoo
SPRING_DATASOURCE_PASSWORD=change-me

AUTH_JWT_SECRET=change-me-base64-secret
VERIFICATION_HASH_SALT=change-me-random-salt
```

## 권장

```env
SPRING_JPA_HIBERNATE_DDL_AUTO=validate
SPRING_JPA_SHOW_SQL=true
SPRING_SQL_INIT_MODE=never
SPRING_SQL_INIT_SCHEMA_LOCATIONS=optional:classpath:db/password_reset_token.sql

AUTH_JWT_ISSUER=pupoo
AUTH_ACCESS_TTL_SECONDS=3600
AUTH_REFRESH_TTL_SECONDS=1209600
AUTH_REFRESH_COOKIE_MAX_AGE_SECONDS=1209600
AUTH_REFRESH_COOKIE_SECURE=false
AUTH_REFRESH_COOKIE_HTTP_ONLY=true
AUTH_REFRESH_COOKIE_SAME_SITE=Lax
AUTH_REFRESH_COOKIE_PATH=/api/auth

APP_CORS_ALLOWED_ORIGIN_PATTERNS=http://localhost:5173,http://localhost:8080,https://pupoo.site,https://www.pupoo.site
```

## Storage SSOT

`app.storage.*`가 SSOT다. compose에서는 아래 `APP_STORAGE_*` 사용을 우선한다.

```env
APP_STORAGE_MODE=local
APP_STORAGE_BASE_PATH=./uploads
APP_STORAGE_PUBLIC_PREFIX=/uploads
APP_STORAGE_KEY_PREFIX=uploads
APP_STORAGE_PUBLIC_BASE_URL=http://localhost:8080
APP_STORAGE_CDN_BASE_URL=https://cdn.pupoo.site
APP_STORAGE_BUCKET=
APP_STORAGE_REGION=ap-northeast-2
APP_STORAGE_ACCESS_KEY_ID=
APP_STORAGE_SECRET_ACCESS_KEY=
APP_STORAGE_SESSION_TOKEN=
```

로컬 파일 스토리지 대신 S3를 쓰는 경우:

```env
APP_STORAGE_MODE=s3
APP_STORAGE_KEY_PREFIX=uploads
APP_STORAGE_PUBLIC_BASE_URL=
APP_STORAGE_CDN_BASE_URL=https://cdn.pupoo.site
APP_STORAGE_BUCKET=change-me
APP_STORAGE_REGION=ap-northeast-2
APP_STORAGE_ACCESS_KEY_ID=change-me
APP_STORAGE_SECRET_ACCESS_KEY=change-me
APP_STORAGE_SESSION_TOKEN=
```

현재 코드에서 legacy alias도 허용한다.

```env
STORAGE_TYPE=local
STORAGE_MODE=local
STORAGE_BASE_PATH=./uploads
STORAGE_PUBLIC_PREFIX=/uploads
STORAGE_KEY_PREFIX=uploads
STORAGE_PUBLIC_BASE_URL=http://localhost:8080
STORAGE_BUCKET=
STORAGE_REGION=ap-northeast-2
STORAGE_ACCESS_KEY_ID=
STORAGE_SECRET_ACCESS_KEY=
STORAGE_SESSION_TOKEN=
```

## 인증/외부 연동

```env
KAKAO_OAUTH_CLIENT_ID=
KAKAO_OAUTH_CLIENT_SECRET=
KAKAO_OAUTH_REDIRECT_URI=https://www.pupoo.site/auth/kakao/callback

KAKAOPAY_BASE_URL=https://open-api.kakaopay.com
KAKAOPAY_SECRET_KEY=
KAKAOPAY_AUTH_PREFIX=SECRET_KEY 
KAKAOPAY_CID=TC0ONETIME
KAKAOPAY_READY_PATH=/online/v1/payment/ready
KAKAOPAY_APPROVE_PATH=/online/v1/payment/approve
KAKAOPAY_CANCEL_PATH=/online/v1/payment/cancel
KAKAOPAY_APPROVAL_URL=http://localhost:8080/payment/approve?paymentId={paymentId}
KAKAOPAY_CANCEL_URL=http://localhost:8080/payment/cancel?paymentId={paymentId}
KAKAOPAY_FAIL_URL=http://localhost:8080/payment/fail?paymentId={paymentId}
```
